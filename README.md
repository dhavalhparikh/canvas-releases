# CANvas — CAN Bus Analyzer for Engineers

A fast, modern CAN bus log visualization and analysis tool for embedded and automotive engineers. Free, open source, and AI-powered.

**[getcanvas.app](https://getcanvas.app)**

## What CANvas does

| | |
|---|---|
| **Log formats** | `.asc`, `.blf`, `.mf4` / `.mdf`, `.csv` |
| **DBC decoding** | Full signal decoding — messages, units, scaling |
| **Interactive plots** | Zoom, pan, linked axes, crosshair, cursor timestamp |
| **Signal panel** | Searchable tree, color-coded per signal, unit labels |
| **Raw frame table** | All frames with decoded signal preview |
| **AI assistant** | Claude, OpenAI, Gemini — bring your own API key |
| **Parse error panel** | Per-frame decode errors surfaced with context |
| **CSV export** | Export selected signals with timestamps |

## Installation

1. Download the zip for your platform from the [Releases](https://github.com/dhavalhparikh/canvas-releases/releases) page
2. Extract the archive
3. Run `CANvas` (Linux / macOS) or `CANvas.exe` (Windows)

No Python installation required — CANvas ships as a self-contained bundle.
